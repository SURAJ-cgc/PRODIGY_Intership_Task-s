```bash

```

### Menu Options:
1. **Encrypt** - Caesar Cipher with optional Base64 encoding
2. **Decrypt** - Caesar Cipher with auto Base64 detection
3. **Base64 Decoder** - Standalone Base64 decoding tool
4. **Exit** - Quit the program
